快速切换vscode代理的工具(vpn和默认之间的切换)
用法就是如果你有魔法的启动工具(exe文件),那么你可以在exe文件的:startfq后加入python D:\ChromeGo\vscode_proxy.py on
地址根据vscode_proxy.py位置而定.
这样启动魔法就自动设置vscode代理了.
关闭的cmd建议放桌面吧,点击后vscode代理恢复默认.当然你也可以把设置Proxy也放桌面

ps:为什么做这个?因为代理改了后出现不上魔法扩展库网络没法用,stm32无法调试,上传git失败等问题,而接入ai(如gemini)又必须要
开魔法.在vscode改设置太麻烦了,现在只需要关闭时点击即可.
